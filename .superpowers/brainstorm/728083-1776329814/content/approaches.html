How should the task detail open?

3 approaches — pick the one that feels right. I've marked my recommendation.

A — Slide-over panel

Panel slides in from the right. Task list stays visible on the left (dimmed). Good if you want context while editing.

Pros

  • List context stays visible
  • Feels modern (Linear, Notion style)

Cons

  • Cramped on mobile
  • More complex animation
RECOMMENDED

B — Centered modal ⭐

Standard centered dialog with backdrop blur. Matches the existing delete confirm modal pattern exactly.

Pros

  • Matches existing patterns
  • Works on mobile
  • Clean focus on the task

Cons

  • Hides the task list

C — Inline expand

The card expands in place. No overlay. Other cards stay visible above and below.

Pros

  • No context switch
  • No overlay to dismiss

Cons

  • Awkward in kanban view
  • No consistent position for edits
  • Shifts layout
