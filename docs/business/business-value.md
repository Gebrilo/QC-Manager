# Business Value

## Value Proposition

QC-Manager transforms quality management from reactive, spreadsheet-driven processes into a data-driven, auditable platform.

## Key Business Outcomes

| Outcome | How QC-Manager Enables It |
|---------|--------------------------|
| **Faster release decisions** | Real-time quality dashboards with pass rates, coverage, and execution freshness |
| **Objective governance** | Configurable quality gates eliminate subjective "gut feel" decisions |
| **Complete auditability** | Every change logged with before/after state; full traceability from bug to test case to story |
| **Reduced defect escape** | Standardized test execution, quality thresholds, and regression tracking |
| **Resource visibility** | Workload dashboards, estimate accuracy tracking, capacity planning |
| **Standardized onboarding** | Journey templates and IDPs ensure consistent new-hire experience |

## Metrics Tracked

| Category | Metrics |
|----------|---------|
| Test Quality | Pass rate, coverage %, execution freshness, blocked tests |
| Bug Health | Bug count by severity, source classification, resolution time |
| Governance | Quality gate status, release readiness, approval history |
| Resource | Estimate accuracy, utilization %, workload distribution |

## Integration Value

- **Tuleap sync**: No dual data entry — artifacts flow bidirectionally between Tuleap and QC-Manager
- **n8n automation**: Scheduled reports, webhook-driven workflows, AI content generation
- **Supabase**: Managed PostgreSQL eliminates self-hosted database maintenance for production
